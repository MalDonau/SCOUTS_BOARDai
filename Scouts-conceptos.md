# 🎨 Scouts — Fichas de concepto

**Categoría:** Piloto (set inicial en diseño activo)
**Scouts:** Deportes · Música · Comida

Las tres respetan el marco de marca:
- **Modern Vintage** (filtro creativo) — analógico, imperfecto, cálido, con textura.
- **Color plano**, sin gradientes ni tints.
- **Keyline 3–5pt** (schematic, no "coloring book").
- Rango **Singular** — figura aislada, legible a **24–32px** (UI-first).
- Motivo del **"!" geekificado** (YEP): una base que toma la vibra de cada interés.

---

## 🏀 Deportes

| | |
|---|---|
| **Color base** | Brick Red `#D44E2D` |
| **Acentos** | Yellow `#F8FB4C`, Black (keyline) |
| **Voz** | Energético · competitivo · *heroic* (Real + Unconventional al frente) |

**Motivo visual:** el `!` como **pelota + trazo de movimiento**. El punto se vuelve la pelota (costuras estilo básquet/cuero), la barra es la estela de impulso. Vintage sport-poster: contraste alto, halftone sutil, un *whoosh* analógico.

**Texturas:** Thick Contrast + Ink (grano de cartel deportivo de los 90).

**Estados**
- **Activo** — salto / impulso hacia arriba, estela marcada.
- **Idle** — pelota rebotando suave, microbote.
- **Observando** — inclinado siguiendo la jugada (la barra apunta).
- **Celebrando** — "brazos arriba": la estela se abre en V, chispa amarilla.

**Legibilidad 24–32px:** queda **pelota + barra inclinada**; descartar costuras finas, mantener silueta y el rojo.

## 🎵 Música

| | |
|---|---|
| **Color base** | Lavender `#A981E9` |
| **Acentos** | Purple `#7D2EFF`, Cyan `#4BC1E9` |
| **Voz** | Rítmico · con vibra · expresivo (Unconventional al frente) |

**Motivo visual:** el `!` como **nota + onda sonora**. La barra se convierte en una **onda / groove de vinilo** que ondula, el punto es la cabeza de nota (o el centro del disco). Guiño directo al vinilo "warm" de los guidelines.

**Texturas:** Offset (desalineación juguetona, tipo impresión de tapa de disco) + Tone on Tone.

**Estados**
- **Activo** — la onda "suena": amplitud máxima, vibrando.
- **Idle** — onda en reposo, línea casi plana con un latido.
- **Observando** — la onda "escucha", se curva hacia un lado.
- **Celebrando** — la onda estalla en ritmo, nota saltando (jumping baseline).

**Legibilidad 24–32px:** reducir a **una sola cresta de onda + nota**; la ondulación fina se simplifica a una curva.

## 🍔 Comida

| | |
|---|---|
| **Color base** | Brown `#8F451D` |
| **Acentos** | Yolk `#E6B30B`, Brick Red `#D44E2D` |
| **Voz** | Cálido · casero · acogedor (Helpful + Real al frente) |

**Motivo visual:** el `!` como **vapor + plato**. La barra es una **voluta de vapor** que sube (de café o comida caliente), el punto es un **plato / moneda cálida** visto desde arriba. Paleta terrosa "perfectly worn-in", el lado más hogareño del sistema.

**Texturas:** Ink + Printing (cross-hatch tipo recetario vintage).

**Estados**
- **Activo** — vapor subiendo con fuerza, "recién servido".
- **Idle** — hilito de vapor suave.
- **Observando** — el vapor se ladea, "olfateando".
- **Celebrando** — el vapor forma un pequeño corazón / chispa amarilla.

**Legibilidad 24–32px:** queda **voluta + plato**; el cross-hatch se omite, se mantiene la curva del vapor y el marrón.
